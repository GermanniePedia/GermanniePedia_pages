张曼妮粉丝后援会

发展历程

缘起

2020年9月，一群热爱整活，精力旺盛的精神少年来到了嘉陵江路中学（42）班。
开学第一天的第三节课，一名年富力强（话说这个词真的可以这样用吗）、英姿飒爽的女老师迈着六亲不认轻快生风的步伐走进了教室。
她就是（42）班同学接下来三年初中生活的主角之一——♥张曼妮♥老师。
见面伊始，有些善于察言观色（想喝茶颜悦色，谁能v我￥30）的同学便从这个老师快速的语流、眼镜片后犀利的眼神里发现这个老师的“难对付”（的确难对付，但也非常可爱！✌）。

成立

2021年，STY同学与SL同学、LXR同学联合创办张曼妮粉丝后援会。
2021年12月，张会第一次全体成员大会召开，此时共有5位成员。

发展

2022年10月开始，由于疫情防控需要，开始居家学习，张会迅速发展，2023年春开学后，成员已有20人（含编外）。
2023年4月，张曼妮老师因病请假近一个月，张曼妮粉丝后援会持续发挥作用，引领班级英语学习劲头，在欢迎张老师归来的大会上，张会起草的欢迎信件，有超过30名同学联署（签名）。

改组

2023年5月26日，张曼妮粉丝后援会与韦张粉丝后援会改组合并为两张粉丝联合会。

主要思想

核心思想：坚决听从张曼妮老师的一切主张，坚决维护张曼妮老师的一切行为，坚定不移跟随张曼妮老师的一切步伐（“三坚”）

发展目标：将英语学科学习透彻，将英语老师喜爱到底。

组织架构

  1. 本人代表处：由张曼妮本人组成，仅一人（名义最高机构）（改组后为张曼妮与韦张两人）
  2. 统战部（实际最高机构）
  3. 应援处（最忙的🤣）
  4. 组织部（关键时候写个文件改个组啥的） 原则上除本人代表处为最高机构以外，其他两部一处地位平等。
